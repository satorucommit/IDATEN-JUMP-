## 🏍️ PROJECT OVERVIEW

**Idaten Jump** is a vibrant, retro-styled fan site dedicated to the classic anime series about mountain bike racing in a parallel dimension called the **X-Zone**. This project showcases epic riders, legendary bikes, and intense MTB battles with a bold neo-design aesthetic.

### 🎨 Design Philosophy
- **Retro Futurism**: Space age vibes meets classic web design
- **Neo-Brutalism**: Bold borders, stark shadows, anti-smooth aesthetics
- **High Energy**: Bright colors, confident typography, playful interactions
- **Anime Tribute**: Capturing the spirit of 2006 anime culture

## 📝 CUSTOMIZATION

### Add More Characters
Edit the **All Characters** section in `index.html` and add new `.char-card` divs.

### Add More Bikes
Edit the **The Bikes** section and add new `.bike-card` divs.

### Change Colors
Update CSS variables in `:root` in `style.css`:
```css
:root {
    --bg-color: #ffffff;
    --text-color: #000000;
    --border-width: 3px;
    --shadow-offset: 6px;
}
```

### Update Images
Replace image paths in HTML with your own images from the `/images/` folder.
